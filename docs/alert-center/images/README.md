## Quy ước

- Ưu tiên **PNG** hoặc **WebP**, độ rộng hợp lý (ví dụ 1200–1600px) để chữ trong ảnh đọc được.
- Che **token / URL đầy đủ** nếu ảnh dùng công khai; có thể để một phần đường dẫn để nhận diện định dạng webhook.
