I. HƯỚNG DẪN CÁCH LẤY 4 THÔNG SỐ TRÊN META
========================================================================

─── BƯỚC 1: LẤY META APP ID & APP SECRET ───

Thực hiện trên: https://developers.facebook.com

  1. Đăng nhập vào Meta for Developers bằng tài khoản Facebook cá nhân.
  2. Chọn App đang sử dụng:
     - Click "My Apps" ở góc trên phải.
     - Chọn App đang dùng để chạy quảng cáo từ danh sách.
  3. Lấy App ID:
     - Vào App Dashboard → App ID hiển thị ở đầu trang (dãy số).
     - Hoặc vào Settings → Basic → trường "App ID".
  4. Lấy App Secret:
     - Vào Settings → Basic → trường "App Secret"
     - Click "Show" để hiển thị → Copy lại.
  5. Kiểm tra App đang ở chế độ Live:
     - Ở thanh trên cùng, xác nhận toggle đang ở trạng thái "Live".
     - Nếu đang ở "Development", chuyển sang "Live".

  ⚠️ LƯU Ý: App phải ở chế độ LIVE thì token mới hoạt động
     trong môi trường production. App ở chế độ Development
     sẽ chỉ hoạt động với admin/developer/tester của app đó.


─── BƯỚC 2: LẤY META BUSINESS ID ───

Thực hiện trên: https://business.facebook.com

  1. Đăng nhập vào Meta Business Manager.
  2. Vào Business Settings (Cài đặt doanh nghiệp):
     - Click icon ⚙️ (Settings) ở menu bên trái.
  3. Vào mục "Business Info" (Thông tin doanh nghiệp):
     - Business ID hiển thị ngay ở mục "Business Info"
       (dãy số dài, ví dụ: 123456789012345).
     - Copy Business ID.


─── BƯỚC 3: TẠO ACCESS TOKEN ───

Thực hiện trên: https://business.facebook.com

  *** Khuyến nghị: Sử dụng System User Token cho production ***

  1. Trong Business Settings, vào menu trái:
     Users → System Users
  2. Tạo System User mới (nếu chưa có):
     - Click "Add" → Đặt tên → Chọn Role: "Admin" → Create.
  3. Gán quyền cho System User:
     - Chọn System User vừa tạo → Click "Add Assets"
     - Gán Ad Accounts cần truy cập → tick "Full Control" → Save.
  4. Generate Access Token:
     - Click "Generate New Token"
     - Chọn App (app đã tạo ở Bước 1)
     - Tick chọn các permission sau:
       ☑️ ads_management
       ☑️ ads_read
       ☑️ pages_show_list      (nếu cần chọn Fanpage khi tạo ads)
       ☑️ pages_read_engagement (nếu cần đọc nội dung Fanpage)
     - Click "Generate Token"
  5. Copy Access Token ngay lập tức:
     - Token chỉ hiển thị 1 lần. Nếu đóng popup mà chưa copy,
       phải generate lại token mới.

  ⚠️ LƯU Ý:
  - System User Token KHÔNG hết hạn (permanent token), trừ khi bị
    revoke thủ công hoặc App bị disable.
  - Nên dùng System User loại Admin để có đầy đủ quyền.
  - Đảm bảo System User đã được gán quyền trên các Ad Account
    cần sync dữ liệu.

I. HOW TO OBTAIN THE 4 PARAMETERS ON META
========================================================================

--- STEP 1: GET META APP ID & APP SECRET ---

Performed on: https://developers.facebook.com

  1. Log in to Meta for Developers using your personal Facebook account.
  2. Select your existing App:
     - Click "My Apps" in the top-right corner.
     - Select the App currently used for running ads from the list.
  3. Get the App ID:
     - Go to App Dashboard → App ID is displayed at the top of the page (numeric string).
     - Or go to Settings → Basic → "App ID" field.
  4. Get the App Secret:
     - Go to Settings → Basic → "App Secret" field
     - Click "Show" to reveal → Copy it.
  5. Verify the App is in Live mode:
     - At the top bar, confirm the toggle is set to "Live".
     - If it is in "Development", switch it to "Live".

  ⚠️ NOTE: The App must be in LIVE mode for the token to work
     in production. An App in Development mode will only work
     for the app's admin/developer/tester.


--- STEP 2: GET META BUSINESS ID ---

Performed on: https://business.facebook.com

  1. Log in to Meta Business Manager.
  2. Go to Business Settings:
     - Click the ⚙️ (Settings) icon in the left menu.
  3. Go to "Business Info":
     - The Business ID is displayed under "Business Info"
       (a long numeric string, e.g., 123456789012345).
     - Copy the Business ID.


--- STEP 3: CREATE AN ACCESS TOKEN ---

Performed on: https://business.facebook.com

  *** Recommended: Use a System User Token for production ***

  1. In Business Settings, go to the left menu:
     Users → System Users
  2. Create a new System User (if one doesn't exist):
     - Click "Add" → Enter a name → Select Role: "Admin" → Create.
  3. Assign permissions to the System User:
     - Select the System User you just created → Click "Add Assets"
     - Assign the Ad Accounts to access → check "Full Control" → Save.
  4. Generate an Access Token:
     - Click "Generate New Token"
     - Select the App (the app created in Step 1)
     - Check the following permissions:
       ☑️ ads_management
       ☑️ ads_read
       ☑️ pages_show_list      (if you need to select a Fanpage when creating ads)
       ☑️ pages_read_engagement (if you need to read Fanpage content)
     - Click "Generate Token"
  5. Copy the Access Token immediately:
     - The token is only displayed once. If you close the popup without
       copying it, you will need to generate a new token.

  ⚠️ NOTE:
  - System User Tokens DO NOT expire (permanent token), unless
    manually revoked or the App is disabled.
  - It is recommended to use an Admin-level System User for full permissions.
  - Make sure the System User has been assigned permissions on the
    Ad Accounts that need data syncing.
